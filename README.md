# Sonic Memoir - Spotify Editorial Showcase

A visually stunning "Audio Visual Journal" inspired by Spotify's editorial design. Built with React, TypeScript, and Tailwind CSS v4.

## Features & Demos

### 1. Hero & Navigation
An immersive hero section with dynamic background animations, floating elements, and a smooth-scrolling navigation system.

![Hero & Navigation Demo](./public/demos/hero_nav.webp)

### 2. Top Tracks & Preview Player
Explore top tracks with hover effects and a context-aware preview player that appears when you play a track.

![Top Tracks & Player Demo](./public/demos/tracks_player.webp)

### 3. Playlist Collections & Detail View
Browse curated playlist collections with "torn paper" aesthetics. Click to reveal detailed playlist views with smooth transitions.

![Playlist Flow Demo](./public/demos/playlist_flow.webp)

## Tech Stack

- **Framework**: React + Vite
- **Language**: TypeScript
- **Styling**: Tailwind CSS v4 (with `@theme` and custom utilities)
- **Icons**: Iconify

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```
