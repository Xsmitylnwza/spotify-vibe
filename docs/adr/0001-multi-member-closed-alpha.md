# Build a multi-member closed alpha

The first release will model Members and Taste Profiles as a multi-member product, but Spotify connection will be invite-only and limited to five Members while the app remains in Development Mode. This requires identity and per-Member data isolation earlier than a personal prototype, but avoids a later rewrite and still allows unlimited Visitors to view public Taste Profiles.
